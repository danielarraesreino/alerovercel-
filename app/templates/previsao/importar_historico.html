Importar Histórico de Vendas

Importe dados de vendas a partir de um arquivo CSV para alimentar o sistema de previsão

Clique para selecionar ou arraste o arquivo aqui

Formato: CSV (valores separados por vírgula)

Formato esperado do arquivo

O arquivo CSV deve conter as seguintes colunas (na ordem):

data,nome_item,quantidade,valor_unitario,periodo_dia,clima,temperatura,evento_especial
2025-05-10,Filé à Parmegiana,5,45.90,Almoço,Ensolarado,28.5,
2025-05-10,Risoto de Camarão,3,59.90,Jantar,Chuvoso,22.0,
2025-05-11,Feijoada,8,39.90,Almoço,Ensolarado,30.0,Final de Semana

Observações:

  • A data deve estar no formato AAAA-MM-DD
  • O nome do item deve corresponder exatamente ao nome do prato ou item de cardápio no sistema
  • Os campos período_dia, clima, temperatura e evento_especial são opcionais
